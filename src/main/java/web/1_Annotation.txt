> SPRING WEB
    > @Controller
        > SPRING MVC 에서 클라이언트의 요청을 처리하고 클라이언트에게 응답을 하는 상황
        > 요청 매핑 : @Controller 크랠스 내 @XXXXMapping 어노테이션들의 특정 URL 경로와 메소드를 매핑 / 연결한다

        > 반환 종류
            > 1. 모델 / 값 반환(@ResponseBody), 주로 AJAX 와 통신할 때 사용
            > 2. 뷰 반환( Templates 는 렌더링 후 클라이언트에게 응답), 주로 페이지 요청할 때 사용

    > @RestController
        > @Controller + @ResponseBody 조합된 컨트롤러 역할
        > 해당 컨트롤 안에 있는 메소드들은 모드 @ResponseBody 역할을 하게 된다.

    > @RequestMapping("/")
        > 클래스 위에 사용 시, 클래스 내 모든 메소드 매핑 URL 의 공통 URL 적용

    > @PostMapping("/")
        > 주로 메소드 위에 사용, 메소드와 매핑할 HTTP 의 URL 정의하고 HTTP POST 방식의 METHOD 사용
